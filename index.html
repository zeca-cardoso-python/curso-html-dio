Aula 1: Formatação de Texto e Estrutura Básica

<h1> até <h6>: São tags de cabeçalhos, usadas para definir títulos de diferentes níveis. A tag <h1> é o título de maior destaque, e as tags subsequentes <h2>, <h3> até <h6> vão diminuindo em tamanho e importância.

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Título Secundário</h3>

<p>: A tag <p> define um parágrafo de texto. É uma das tags mais usadas em HTML.

<p>Este é um parágrafo de exemplo</p>

<mark>: Destaca o texto, geralmente usado para marcar partes do conteúdo que são importantes ou foram encontradas em uma busca.

<mark>Texto destacado</mark>

<small>: Usado para diminuir o tamanho do texto, geralmente para citações ou detalhes menores.

<small>Texto em tamanho menor</small>

<i>: Aplica o estilo itálico no texto, geralmente usado para ênfases.

<i>Texto em itálico</i>

<u>: Aplica o estilo sublinhado ao texto.

<u>Texto sublinhado</u>

<strong>: Usada para enfatizar o texto, geralmente exibido em negrito. Ao contrário da tag <b>, <strong> tem um valor semântico de importância.

<strong>Texto em negrito</strong>

Aula 2: Listas e Links

<ol>: Cria uma lista ordenada, ou seja, uma lista onde os itens são numerados.

<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

<ul>: Cria uma lista não ordenada, onde os itens são precedidos por marcadores (pontos).

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<li>: Define um item dentro de uma lista, seja ela ordenada (<ol>) ou não ordenada (<ul>).

<li>Este é um item</li>

<a>: A tag de link, usada para criar links que direcionam o usuário a outra página ou recurso na web.

<a href="https://www.exemplo.com">Visite o site</a>

<hr>: Cria uma linha horizontal para separar seções do conteúdo. Normalmente usado para dividir conteúdo de forma visual.

<hr>

Aula 3: Tags Matemáticas e de Referência

<sub>: A tag <sub> coloca o texto em um formato de subscrito, geralmente usado para notações científicas ou fórmulas químicas.

<sub>Texto subscrito</sub>

<sup>: A tag <sup> coloca o texto em um formato de sobrescrito, geralmente usado para exponentes ou notações matemáticas.

<sup>Texto sobrescrito</sup>

<blockquote>: A tag <blockquote> é usada para indicar citações longas, geralmente com um recuo na margem.

<blockquote>Esta é uma citação longa</blockquote>

<br>: A tag <br> é usada para quebrar uma linha dentro de um parágrafo ou outro elemento de bloco.

Texto linha 1<br>Texto linha 2

Aula 4: Tags de Estilo e Formatação Adicional

<font>: A tag <font> era usada para alterar o estilo da fonte (como cor, tamanho e tipo), mas foi descontinuada e não é mais recomendada para uso.

<font color="red">Texto em vermelho</font>

<del>: A tag <del> é usada para marcar um texto que foi deletado. Normalmente é exibido com uma linha sobre o texto.

<del>Texto deletado</del>

<abbr>: A tag <abbr> é usada para abreviar uma palavra ou expressão e deve conter o atributo title com a forma completa.

<abbr title="HyperText Markup Language">HTML</abbr>
